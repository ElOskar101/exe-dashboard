# Template

Minimal React + TypeScript + Vite starter with Tailwind dark/light mode and socket support.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## E2E tests

Create `.env.e2e.local` with the test login credentials:

```bash
E2E_TEST_USERNAME=
E2E_TEST_PASSWORD=
```

Then run:

```bash
pnpm run test:e2e
```

## Structure
